Bienvenidos a mi pagina web

Esta vez estare viendo "Listas Ordenadas" y "Listas Desordenadas"

Primero veremos listas ordenadas

Hagamos una "lista ordenada"(ol) de frutas:

  1. Manzanas
  2. Peras
  3. Naranjas
  4. Mandarinas
  5. Kiwis
Como se puede ver, la lista esta ordenada simplemente de forma numerica.

Ahora hagamos una "lista desordenada"(ul) de frutas:

  • Manzanas
  • Peras
  • Naranjas
  • Mandarinas
  • Kiwis
Como se puede ver, es simplemente una lista, tiene ningun tipo de orden..

Ahora vamos hacer una lista de frutas, cun una sublista.

  • Manzanas
    • Rojas
    • Verdes
  • Peras
    • Amarillas
    • Verdes
  • Naranjas
  • Mandarinas
  • Kiwis
Este tipo de sublistas se pueden hacer tambien con listas del tipo Ordenadas.
  1. Manzanas
    1. Rojas
    2. Verdes
  2. Peras
    1. Amarillas
    2. Verdes
  3. Naranjas
  4. Mandarinas
  5. Kiwis
